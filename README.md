# 🎲 Dice Game

A simple and fun Dice Game built using **HTML, CSS, and JavaScript**.

Just refresh the page and see who wins! 🎉

---

## 📌 Project Description

This is a two-player dice game.  
Whenever the page is refreshed, both players get a random dice number.  
The player with the **higher dice number wins**.

If both players get the same number, the game shows **Draw**.

---

## 🕹️ How to Play

1. Open the game in your browser  
2. Refresh the page 🔄  
3. Dice numbers are generated automatically  
4. The player with the higher number wins  

---

## 🧠 Game Rules

- Player 1 and Player 2 roll one dice each
- Dice numbers are generated randomly
- Higher number = Winner
- Same number = Draw 🤝
